What is Web Development?

Web development is the process of building and maintaining websites and web applications that run in a web browser.

Examples of Web Development

  • Building an e-commerce site like Amazon
  • Developing a social media app like Instagram
  • Creating an internal dashboard for a business

Three Main Areas of Web Development

  1. Front-end Development
  2. Back-end Development
  3. Full-stack Development

• HTML

HTML stands for HyperText Markup Language. It's the standard language used to create and structure content on the web.

• CSS

CSS stands for Cascading Style Sheets. It's the language used to style and lay out HTML content on a web page.

• JavaScript

JavaScript is a programming language used to make websites interactive. While HTML structures a web page and CSS styles it, JavaScript makes it do things like respond to user actions (clicks typing, etc.), show/hide content, animate elements, etc.

• JQuery

jQuery is a fast, small, and feature-rich JavaScript library that simplifies common tasks like DOM manipulation, event handling, animations, and AJAX calls. It was created to make JavaScript easier to write and work consistently across different browsers.

• Bootstrap

Bootstrap is an open-source front-end framework used to build responsive websites quickly. It provides pre-designed HTML, CSS, and JavaScript components that help developers create clean and functional user interfaces without writing everything from scratch.

• DOM

The DOM stands for Document Object Model. It's a programming interface for web pages. The DOM is how your browser represents the HTML page in memory, so that JavaScript (or other code) can read or change it.

• Unix Command

Unix commands are text-based instructions you type into a Unix shell (a command-line interface) to interact with a Unix or Unix-like operating system (like Linux or macOS).

• GitHub

GitHub is a platform for hosting and collaborating on code projects using Git, a version control system. It helps developers store their code online, track changes, work together with others.


Steps for Web Development

1. Choosing Your Code Editor

Visual Studio Code is a powerful code editor developed by Microsoft. It is popular among developers for writing code in many programming languages, including HTML, CSS, JavaScript, Python, and more.

2. Wireframe

Wireframe is a simple visual guide or blueprint that outlines the structure and layout of a web page or app before the actual design and development start.

3. Writing Codes

Here are steps to follow; open a code editor, create a new file called index.html, write your code, save and open the file in a web browser, try editing the text, colors, or JavaScript function to experiment.

4. Building Responsive Websites

Building responsive websites means designing and coding your site so it looks great and works well on all devices — from huge desktop monitors to tiny smartphones.

5. Tests

Testing is a step to make sure it works correctly, is free of bugs, and makes refactoring safer.

6. Deployment

Deployment in web development means publishing your website or web app so that anyone on the internet can access it.

Visual Studio Code - Extentions

  • vscode-icons

    vscode-icons is a popular Visual Studio Code extension that adds file icons to your editor’s file explorer. You can customize icon visibility and styles through settings.

  • GitHub Theme

    GitHub Theme usually refers to a Visual Studio Code theme inspired by GitHub’s look, or even the official GitHub theme extension that makes your VS Code editor look like GitHub’s UI.

  • Live Server

    Live Server is a super popular Visual Studio Code extension that lets you launch a local development server with live reload capability.


Markup Validation

Markup Validation is the process of checking your HTML (or other markup languages like XHTML) to ensure your code follows the official syntax rules and standards. This helps your website work correctly across browsers and improves accessibility and SEO.

Tools for Wireframe

  • Balsamiq

    Balsamiq is a popular wireframing tool designed to help you quickly create low-fidelity, sketch-like mockups of user interfaces for websites, apps, or software. It is a drag-and-drop wireframing app with a focus on simplicity and speed. Ba;sa,oq makes wireframes look like hand-drawn sketches to emphasize structure and flow, not polished design. It helps teams brainstorm, plan layouts, and share early concepts before development.

  • Figma

    Figma is a powerful, web-based UI/UX design and prototyping tool that’s hugely popular for creating everything from wireframes to fully polished designs. It is a collaborative design platform that works right in your browser (no heavy installs).It lets multiple people work together in real time on the same design file. Figma supports wireframing, UI design, prototyping, and even developer handoff. You can create interactive prototypes to simulate user flows and transitions.

  • Sketch

    Sketch is a popular vector-based design tool primarily used for UI/UX design, especially on macOS. It’s widely loved by designers for creating wireframes, mockups, and prototypes for websites and apps. It is a macOS-only application for designing interfaces and digital products. Sketch focuses on vector graphics, meaning your designs stay sharp at any size. It is known for its simple, intuitive interface tailored for designers. It supports symbols (reusable design elements), artboards, and plugins.

  • Adobe XD

    Adobe XD is a powerful design and prototyping tool from Adobe, widely used for creating user interfaces, wireframes, and interactive prototypes for websites and apps. It is a vector-based design tool focused on UI/UX design.Adobe XD allows designers to create wireframes, high-fidelity mockups, and interactive prototypes.It supports collaboration and sharing with stakeholders and is available on Windows and macOS.

  • Others

    There are other useful wireframe tools you may consider: Axure RP, Wireframe.cc, Mockflow, etc.
